Полезные ссылки:

[Документация CatBoost](https://catboost.ai/docs/)

## Структура проекта

```text
cyberpolka/
├── baseline_catboost.ipynb       # базовый ноутбук с моделью CatBoost
├── clean_baseline_catboost.ipynb # очищенная и упрощённая версия базового ноутбука
├── eda_extra_features.ipynb      # разведочный анализ данных и генерация дополнительных признаков
├── feature_extract.ipynb         # извлечение и сохранение признаков для обучения моделей
├── extra_eda/                    # дополнительные эксперименты и исследовательские ноутбуки
├── srs/                          # служебные модули и общие импорты (например, imports.py)
├── data/                         # исходные данные и подготовленные датасеты
├── modified_data/                # модифицированные/обработанные данные (feature store, промежуточные версии)
├── models/                       # сохранённые обученные модели
├── models_extraction/            # ноутбуки/скрипты для анализа и извлечения моделей
└── submits/                      # файлы сабмитов для соревнования
```
